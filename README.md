ItsToDoAgain
=============
Yet another todo app on MEAN STACK 
MongoDB [ Mongolab ] 
ExpressJS
AngularJS
NodeJS

### Install

* Download/clone the repo
* Run `npm install`
* Run node server/app.js and navigate to `http://localhost:3000` to view the app
